vkCreateBuffer

函数原型​

VkResult vkCreateBuffer(
    VkDevice                                    device,
    const VkBufferCreateInfo*                   pCreateInfo,
    const VkAllocationCallbacks*                pAllocator,
    VkBuffer*                                   pBuffer);

描述​

创建一个buffer。

参数​

  • device : 用于创建buffer的逻辑设备。
  • pCreateInfo : VkBufferCreateInfo 结构体指针，包含创建buffer的信息。
  • pAllocator : host memory 分配器。
  • pBuffer : 创建返回的buffer句柄。

返回值​

  • VK_SUCCESS : 成功创建image。
  • 其他 : 创建失败。
